# Yandex Scooter API Tests

## Описание проекта
Проект содержит автоматизированные тесты для API Яндекс.Самокат и SQL-запросы для проверки корректности работы базы данных.

## Задания

### 1. Работа с базой данных
**Задание 1:** Вывести список логинов курьеров с количеством их заказов в статусе «В доставке» (поле inDelivery = true).

**Задание 2:** Вывести все трекеры заказов и их статусы согласно правилам:
- finished == true → статус 2
- cancelled == true → статус -1  
- inDelivery == true → статус 1
- остальные случаи → статус 0

### 2. Автоматизация теста API
**Сценарий:** Клиент создает заказ, проверяется, что по треку заказа можно получить данные о заказе.

**Шаги автотеста:**
1. Выполнить запрос на создание заказа
2. Сохранить номер трека заказа
3. Выполнить запрос на получение заказа по треку заказа
4. Проверить, что код ответа равен 200
